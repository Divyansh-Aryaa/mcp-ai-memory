## Development Best Practices

### Daily Workflow
1. Store memories at key milestones during the day
2. Use descriptive text when storing memories
3. Review yesterday's work before starting new tasks
4. Weekly review to identify patterns and focus areas

### Tips
- Use consistent terminology in memory entries
- Include context for better future reference
- Store both what you did AND what you learned
- Regular memory searches help identify repeated work
